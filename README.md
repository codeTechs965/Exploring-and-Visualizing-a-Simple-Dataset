# 🌸 Iris Dataset Analysis (Python EDA Project)

This project performs **Exploratory Data Analysis (EDA)** on the famous Iris dataset using Python.

It includes data inspection, statistical summaries, and multiple visualizations to understand feature distributions and class balance.

---

## 🚀 Features

* 📊 Data inspection (`head`, `info`, `describe`)
* 📈 Class distribution analysis
* 📉 Histogram visualization
* 📦 Box plot for outlier detection
* 🎨 Seaborn-based plots
* 🌸 Species-wise analysis

---

## 🛠️ Tech Stack

* Python 🐍
* pandas → Data handling
* matplotlib → Basic visualization
* seaborn → Advanced visualization

---

## 📂 Dataset

The dataset used is the classic Iris dataset containing:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species (target variable)
